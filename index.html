Limitaciones

  • Requiere Windows 10/11 Pro, Enterprise o Education con Windows Sandbox habilitado.
  • GitHub Pages es estatico: no se ejecuta Sandbox desde la web. Se descarga el pack y el launcher.
  • OK no es garantia absoluta. El contexto y las firmas importan.
  • Sin red, el update puede fallar y el resultado queda inconcluso.

  • Firmas viejas o update fallido.
  • Senales fuertes: MOTW + sin firma + redirecciones + comportamiento anomalo.
  • No se encontro un archivo estable para escanear.
