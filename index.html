Requisitos

Lo mínimo para correr WinLab

Sin esto, el sandbox no abre o queda inconcluso.

  • Windows 10/11 Pro, Enterprise o Education. Windows Sandbox no está disponible en Home.
  • Windows Sandbox habilitado. PowerShell admin: Enable-WindowsOptionalFeature -Online -FeatureName Containers-DisposableClientVM -All -NoRestart
  • Permisos para Defender. El escaneo usa Defender. Si no puede actualizar, el resultado puede ser inconcluso.
  • Safe Mode Helper (opcional). Descárgalo para activar modo seguro con guía y confirmación: Ver helper.

FAQ

Respuestas rápidas

Lo esencial antes de comprar o probar.

¿Funciona en Windows Home?+

No. Windows Sandbox requiere Windows 10/11 Pro, Enterprise o Education.

¿WinLab ejecuta el archivo?+

No lo ejecuta. Analiza y reporta. El objetivo es reducir riesgo, no probar malware en el host.

¿Dónde quedan los reportes?+

En %LOCALAPPDATA%\WinLab\outbox si el preset tiene outbox ON. Logs en %LOCALAPPDATA%\WinLab\logs.

¿Qué pasa si sale INCONCLUSO?+

Puede ser por firmas viejas, update fallido o señales fuertes (MOTW + sin firma + red). Repite con red en Balanced/Networked.

¿Puedo analizar un link sospechoso?+

Sí. Usa el preset Networked y modo terminal: WinLab.ps1 url -Url "https://...". El reporte incluye DNS/TLS/headers.

¿Cómo se paga?+

El sitio es estático. Pagas vía WhatsApp/transferencia o el link de pago que configures en assets/config.js. Luego recibes soporte/manual.
