Guia breve para clientes

WinLab analiza archivos o URLs sospechosas dentro de Windows Sandbox y te entrega un reporte claro. Sirve para reducir riesgo sin ensuciar tu PC.

Requisitos

  • Windows 10/11 Pro, Enterprise o Education.
  • Windows Sandbox habilitado.
  • Virtualizacion activa en BIOS/UEFI.

Instalacion

  1. Ejecuta el instalador de WinLab.
  2. Se crean accesos directos en el Escritorio y el Menu Inicio.
  3. Listo: ya podes abrir WinLab.

Si Windows Sandbox no esta habilitado, WinLab te muestra los pasos para activarlo.

Como usarlo

  1. Abr√≠ el launcher de WinLab.
  2. Elegi un preset (Equilibrado, Ultra seguro o Con red).
  3. Arrastra un archivo o pega una URL.
  4. Al finalizar, se abre el reporte automaticamente.

Reportes en C:\WinLab_Outbox. Logs en C:\WinLab\logs.

Diagnostico rapido

Si algo no funciona, ejecuta el acceso directo de WinLab Doctor. Genera un reporte en C:\WinLab\logs\doctor.txt.

Limitaciones

WinLab reduce riesgo, pero no garantiza deteccion total. Si el archivo es critico, valida con otras herramientas.
