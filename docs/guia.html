Guía rápida de WinLab

WinLab usa Windows Sandbox y Microsoft Defender para analizar archivos o URLs sospechosas sin tocar el host. No es un antivirus propio: reduce riesgo, no lo elimina.

Requisitos

  • Windows 10/11 Pro, Enterprise o Education.
  • Windows Sandbox habilitado (Containers-DisposableClientVM).
  • Virtualización habilitada en BIOS/UEFI.

Instalación

  1. Extraé el ZIP completo del producto.
  2. Ejecutá WinLab_Setup.cmd desde la carpeta extraída.
  3. Se crean accesos directos y el lanzador principal.

Si Windows Sandbox no está habilitado, el instalador te ofrece activarlo (requiere permisos de Administrador).

Cómo usar el lanzador

  1. Ejecutá WinLab_Launcher.cmd.
  2. Elegí el perfil (Equilibrado, Ultra seguro o Con red).
  3. Arrastrá un archivo o pegá una URL según corresponda.
  4. Al finalizar, el reporte se abre automáticamente.

Reportes en C:\WinLab_Outbox. Logs en C:\WinLab\logs.

Perfiles

Equilibrado (Balanced): red limitada, actualización de firmas y reporte rápido.

Ultra seguro (UltraSecure): sin red y máxima reducción de superficie.

Con red (Networked): red habilitada para analizar URLs (usalo con cautela).

Modo terminal (opcional)

Si querés automatizar, WinLab incluye CLI:

  • WinLab.ps1 scan -Path "C:\ruta\archivo.ext" -Preset UltraSecure -Minutes 5 -OpenOutbox
  • WinLab.ps1 url -Url "https://ejemplo.com" -Preset Networked -Minutes 15 -OpenOutbox

Reportes

WinLab genera reportes HTML/JSON/TXT con resumen ejecutivo y detalle técnico.

  • OK: no hubo detecciones en este run. No es garantía absoluta.
  • DETECTADO: Microsoft Defender detectó una amenaza.
  • INCONCLUSO: el resultado no es confiable (firmas viejas, red bloqueada o señales fuertes).

Solución de problemas

  • Si Sandbox no abre: habilitá Containers-DisposableClientVM y reiniciá.
  • Si no aparece reporte: revisá C:\WinLab\logs.
  • Si el update falla: reintentá en Equilibrado con red habilitada.

Soporte

Escribinos desde el sitio o por WhatsApp para ayudarte a configurar WinLab y resolver dudas.

Limitaciones

WinLab reduce riesgo, pero no garantiza detección total. Siempre validá por otra vía si el origen es dudoso.
