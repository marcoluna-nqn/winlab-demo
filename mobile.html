Opción 1

Laboratorio Remoto (recomendado)

Una PC segura con Windows dedicado y WinLab, accesible desde el celular por una red VPN cerrada. Sin exponer puertos.

Requisitos Host Windows
  • Windows 10/11 Pro, Enterprise o Education.
  • Windows Sandbox habilitado.
  • Conexión estable a Internet.
  • Cuenta dedicada para el laboratorio.
Pasos claros 10 minutos
  • 1
    Instalá Tailscale en el host y en el celular.
  • 2
    Conectá ambos a la misma red segura (VPN).
  • 3
    Abrí Escritorio remoto de Microsoft y entrá por la IP de Tailscale.
  • 4
    Ejecutá WinLab en el host y generá el reporte.
Seguridad primero No abrir puertos
  • No abras el puerto RDP en el router ni en la nube.
  • Usá Tailscale o una VPN privada y cerrada.
  • Contraseña fuerte y usuario exclusivo para WinLab.
  • Bloqueo de pantalla y actualizaciones al día.
  • Usá BitLocker si tu edición lo permite.
  • Clave API obligatoria en el header X-WINLAB-KEY. No la compartas.

FAQ

Preguntas frecuentes desde el celular

¿Se instala en iPhone o Android?+

No. El sandbox corre en Windows. En el celular solo usás acceso remoto o el servicio asistido.

¿Es seguro conectarme remoto?+

Sí. Siempre que uses VPN tipo Tailscale, sin puertos abiertos y con credenciales fuertes.

¿Qué pasa si el enlace es malicioso?+

Se abre dentro de Windows Sandbox. El entorno se descarta al cerrar.

¿Qué no hace WinLab?+

No reemplaza un antivirus completo ni garantiza detección total. Es una capa de análisis y reporte.
