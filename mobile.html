Opcion 1

Laboratorio Remoto (recomendado)

Un Windows dedicado con WinLab, accesible desde el celular por una red segura tipo VPN. Sin exponer puertos.

Requisitos Host Windows
  • Windows 10/11 Pro, Enterprise o Education.
  • Windows Sandbox habilitado.
  • Conexion estable a Internet.
  • Cuenta dedicada para el laboratorio.
Pasos claros 10 minutos
  • 1
    Instala Tailscale en el host y en el celular.
  • 2
    Conecta ambos a la misma red segura (VPN).
  • 3
    Abri Escritorio remoto de Microsoft y entra por la IP de Tailscale.
  • 4
    Ejecuta WinLab en el host y genera el reporte.
Seguridad primero No abrir puertos
  • No abras el puerto RDP en el router ni en la nube.
  • Usa Tailscale o una VPN privada y cerrada.
  • Contrasena fuerte y usuario exclusivo para WinLab.
  • Bloqueo de pantalla y actualizaciones al dia.
  • Usa BitLocker si tu edicion lo permite.
  • Clave API obligatoria en el header X-WINLAB-KEY. No la compartas.

FAQ

Preguntas frecuentes desde el celular

Se instala en iPhone o Android?+

No. El sandbox corre en Windows. En el celular solo usas acceso remoto o el servicio asistido.

Es seguro conectarme remoto?+

Si. Siempre que uses VPN tipo Tailscale, sin puertos abiertos y con credenciales fuertes.

Que pasa si el enlace es malicioso?+

Se abre dentro de Windows Sandbox. El entorno se descarta al cerrar.

Que no hace WinLab?+

No reemplaza un antivirus completo ni garantiza deteccion total. Es una capa de analisis y reporte.
